GitHub on veebiplatvorm projektide salvestamiseks ja haldamiseks Giti versioonikontrollisüsteemi abil.
Seda kasutavad arendajad, meeskonnad ja organisatsioonid koodi koostööks.
Tulevikus võin GitHubi kasutada hariduslike ja professionaalsete projektide salvestamiseks ning meeskonnatööks.

GitHub Inc. asutasid 2008. aastal Tom Preston-Werner, Chris Wanstrath ja PJ Hyett.
GitHub muutus kättesaadavaks hiljem samal aastal.
